# 표준 내장 객체 (Standaro Built-in Object)는 자바스크립트가 기본적으로
      가지고 있는 객체를 의미.
      
      JavaScript    ->   Js개발자   ->    User
        (기본적인 기능제공 API)    (UI 제공)


      자바스크립트 내장 객체
      -----------------------
      - Object
      - Function
      - Array
      - String
      - Boolean
      - Number
      - Math
      - Date
      - RegExp (정규표현식)

      사용자가 직접 정의하여 사용하는 객체를 [사용자 지정 객체] 라고한다
